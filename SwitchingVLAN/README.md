# Switching VLAN

To practice this example:
- use PacketTracer
- add 3 Cisco 2960 switch
- connect them with crossover cables

## Objective
Connect 2 VLANs

## Commands
`show vlan`

`show int trunk`

`show int <Interface> switchport` (i.e. shot int f0 / 21 switchport)


Using Utiliser Cisco Packet Tracer 6.2.0
